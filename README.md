# Image Slider / Carousel Website

Description:
> This project is a responsive image slider built using HTML, CSS, and JavaScript. It displays multiple images with smooth transitions and allows users to navigate using next/previous buttons and clickable thumbnails.

Key Features:

> Automatic sliding every 5 seconds<br>
Manual navigation (Next/Previous buttons)<br>
Thumbnail-based navigation<br>
Smooth animations and transitions<br>
Responsive design for different screen sizes<br>

Technologies Used:

> HTML (structure)<br>
CSS (styling & animation)<br>
JavaScript (functionality & DOM manipulation)<br>
